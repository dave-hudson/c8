# c8::integer::is\_zero #

```cpp
auto is_zero() const noexcept -> bool;
```

Determine if this object has a value of zero.

### Return Value ###

* `true` if the value of this object is zero.
* `false` if it is not zero.

### Exceptions ###

None.
