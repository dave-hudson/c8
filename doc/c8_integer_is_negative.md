# c8::integer::is\_negative #

```cpp
auto is_negative() const noexcept -> bool;
```

Determine if this object has a value that is negative.

### Return Value ###

* `true` if the value of this object is negative.
* `false` if it is not negative.

### Exceptions ###

None.
